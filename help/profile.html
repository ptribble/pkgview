Jumpstart Profile Builder

The jumpstart profile builder is a graphical utility to select packages for a jumpstart profile.

On the first screen, you must select a starting metacluster, which defines the initial set of packages which you can then customize.

On the package selection tab, you can view packages either alphabetically or hierarchically by cluster.

When viewing a package or cluster, going to the 'Customize' tab allows you to modify which software would be installed by the jumpstart profile.

When you're satisfied, go to the 'Output' tab, where you can show the profile that you've created, and save it to a file.
