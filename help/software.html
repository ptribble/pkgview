Solaris Package Viewer

The Package Viewer allows you to study the SVR4 packages installed on your system.

There are 4 available views:

  • The Package list shows the packages installed, initially in alphabetical order of package name, although you can change the sort order using the column headers.
  • The Tree View shows the packages sorted into clusters in the way that the installer handles them.
Both the above views give general information on the package, the other packages it depends on and which depend on it, and which install clusters and metaclusters the package is part of.
  • The Filesystem view allows you to browse the filesystem and see which packages and install clusters contain a file.
  • The Patches view allows you to see what patches are installed on your system, and get some information about those patches.
